## Hosting on GitHub Pages

1. Push this folder to a GitHub repo (e.g., `<username>.github.io` or any repo)
2. Go to **Settings → Pages → Source** and select the `main` branch, `/ (root)` folder
3. The site will be live at `https://<username>.github.io`

## Updating content

Edit any `.html` file directly — no build step required. Commit and push to update the live site.
